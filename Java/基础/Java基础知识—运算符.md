#### 1. 算术运算符

```java
+,-,*,/,%,++,--
```

#### 2. 关系运算符

```java
>,<,>=,<=,==,!=
```

#### 3. 逻辑运算符
```java
!,&&,&,|,^,||
```


#### 4. 赋值运算符

```java
=,+=,*=,/=,-=,%=
```

#### 5. 位运算符
```java
&,|,^,<<,>>,>>>,~
```


#### 6. 条件运算符

```java
?:
```

-----

#### 运算符优先级表格

| 优先级 | 运算符                                         | 结合性   |
| ------ | ---------------------------------------------- | -------- |
| 1      | ()、[]、{}                                     | 从左向右 |
| 2      | !、+、-、~、++、--                             | 从右向左 |
| 3      | *、/、%                                        | 从左向右 |
| 4      | +、-                                           | 从左向右 |
| 5      | «、»、>>>                                      | 从左向右 |
| 6      | <、<=、>、>=、instanceof                       | 从左向右 |
| 7      | ==、!=                                         | 从左向右 |
| 8      | &                                              | 从左向右 |
| 9      | ^                                              | 从左向右 |
| 10     | \|                                             | 从左向右 |
| 11     | &&                                             | 从左向右 |
| 12     | \|\|                                           | 从左向右 |
| 13     | ?:                                             | 从右向左 |
| 14     | =、+=、-=、*=、/=、&=、=、^=、~=、«=、»=、>>>= | 从右向左 |
